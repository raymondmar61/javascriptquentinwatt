JavaScript is an object based programming language. Everything in JavaScript is an object. Think of an object like an object in real life. For example, a car is an object.

The name lamborgini is a property. The color red is a property.

Method is an action. The car can start, get it and drive, stop the car, park the car are all actions. An action with an object is a method. Methods end with paranthesis (). For example, toUpperCase().

var x = "My name is Quentin";. Variable x is an object. x.length .length is the property.
